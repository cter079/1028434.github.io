Microsoft

Camryn Terlouw

Geschiedenis

Microsoft is opgericht door Bill Gates en (vaak vergeten) Paul Allen in 1975 in New Mexico. Paul Allen richtte het bedrijf op na hij kansen zag liggen om BASIC (een programmeertaal) te implementeren in de eerste microcomputer (Altair 8800). Microsoft kreeg zijn naam door Paul Allen. Hij combineerde “microcomputer” met “software” en noemde dat eerst Micro-Soft.(later door Bill Gates veranderd naar Microsoft). De eerste vestiging van Micro-Soft stond in Japan in 1978. Microsoft begon met producten zoals Microsoft BASIC (programmeertaal) en later ook Xenix(een versie van een besturingssysteem zoals Android of IOS). Op Xenix kwam ook de eerste versie van Multi-tool-Word, wat we nu Word noemen. In 1983 kwam Microsoft met een besturingssysteem voor de Personal Computer(de computer die we thuis kunnen gebruiken):MSX. In 1983 breidde Microsoft ook zijn assortiment uit. Zo kwamen ze met hun eigen muis: Microsoft Mouse. Later, in 1985 kwam de eerste versie van het bekendste besturingssysteem ooit beschikbaar op de markt: Windows 1.0. Later volgde ook OS/2(een ander besturingssyteem) dat voor IBM Personal Computers werd gereleaset. In 1989 kwam het meest bekende en succesvolle product van Microsoft: Microsoft Office. Hierin zaten Microsoft Word(tekstbewerken) en Microsoft Excel(rekentool). In 1990 releasete Microsoft Windows 3.0 die wat aanpassingen had aan het interface(= beelden en menu’s op je computer om hem te bedienen). Windows werd meer succesvol dan OS/2 en dus stopte Bill met het verder ontwikkelen ervan. In 1992 kwam er voor het eerst een reclame op tv over Windows 3.1 en werd in 1993 het meest gebruikte OS op de markt. In 1995 lanceerde Microsoft Windows 95 met een gratis Web Browser: Internet Explorer (voormalig Spyglass). Echter was Internet Explorer alleen te gebruiken in de Plus versie van Windows 95. In hetzelfde jaar lanceerde Microsoft ook het MSN(Microsoft Network) wat de meeste mensen nog kennen. MSN is een webportaal met allerlei soorten apps en diensten in je browser. In 1998 werd Steve Ballmer de nieuwe president van Microsoft, wel bleef Bill Gates de CEO. In hetzelfde jaar werd ook Windows 98 gelanceerd met veel nieuwe toevoegingen. Later in 2001 werd Windows XP gelanceerd met een heel nieuw interface. In hetzelfde jaar breidde Microsoft zijn assortiment uit met een nieuwe game console: XBOX. Deze game console was met verkopen net achter de Playstation geëindigd, maar versloeg wel andere consoles zoals de Game Cube. In 2002 kwam Microsoft met Microsoft Visual Studio(een programma om te leren en te kunnen programmeren) en een nieuwe programmeertaal: C#. In 2006 lanceerde Microsoft de XBOX 360 die het op moest nemen tegen de Playstation 3 en de futuristische Wii met bewegingssensoren. De XBOX 360 kampte met veel problemen zoals de red ring of death dat zorgde voor veel geldverlies. Het jaar erna kwam Microsoft met Windows Vista die ook Office en Visual Studios een update gaf.
